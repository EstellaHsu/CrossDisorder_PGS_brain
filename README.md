# CrossDisorder_PGS_brain
This is the code repository for the manuscript "**Neuroimaging across diverse ancestries links pleiotropic genetic risk for internalizing disorders to adolescent brain development"


Of note, the code of vertex-wise linear mixed models that used in this paper has been upgraded into an R package: **verywise**. Please see the package here: https://github.com/SereDef/verywise
